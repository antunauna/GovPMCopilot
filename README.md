# GovPMCopilot

GovPMCopilot 是一个面向政府项目管理与申报场景的技能集合，覆盖从政策解读、主体匹配、材料撰写到审核复核、流程编排的完整链路。

## 内容结构

| 文件 | 说明 |
|------|------|
| `EntityMatcher.md` | 主体匹配器，负责企业主体资格匹配与推荐。 |
| `PolicyInterpreter.md` | 政策解读器，负责政策解析、申报策略和风险识别。 |
| `ProposalComposer.md` | 材料撰写器，负责申报书、可研报告等材料生成。 |
| `ReviewAuditor.md` | 预审复核器，负责材料质检、问题识别与修改建议。 |
| `WorkflowOrchestrator.md` | 流程编排器，负责全链路任务调度、状态管理与协同。 |

## 适用场景

- 政策申报项目管理
- 项目材料自动化生成
- 政策与企业主体匹配分析
- 申报流程协同与进度追踪
- 预审、复核与合规把控
