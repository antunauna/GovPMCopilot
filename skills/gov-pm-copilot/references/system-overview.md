# System Overview

GovPMCopilot is a composite enterprise skill system for end-to-end government project automation.

## Functional chains

- Policy collection: discover, capture, classify, and maintain policy intelligence.
- Interpretation and evaluation: determine whether a project is worth pursuing and which entity should apply.
- Drafting and synthesis: generate structured application materials from verified inputs.
- Review and remediation: run expert-style pre-review and identify corrective actions.

## Six core skills

- PolicyRadar
- EntityMatcher
- PolicyInterpreter
- ProposalComposer
- ReviewAuditor
- WorkflowOrchestrator

## Non-functional requirements

- Multi-entity support across group companies.
- Authority and source traceability.
- Timeliness and lifecycle tracking.
- Compliance-first generation behavior.
- Standardized outputs for repeatable reuse.
- Extendability to new project lines such as high-tech enterprise, specialization, industrial upgrading, and research grants.
