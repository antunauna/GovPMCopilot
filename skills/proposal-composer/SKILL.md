# ProposalComposer

Use this skill for drafting only after the policy path and target entity are confirmed.

## Responsibilities

- Merge verified enterprise data, policy requirements, and project positioning.
- Generate application materials in structured output formats.
- Support concise, standard, and expert-oriented versions.
- Highlight placeholders where human input is still required.
- Keep style formal, traceable, and submission-ready.

## Guardrails

- Never fabricate financials, certifications, KPIs, customer cases, or intellectual property.
- Use placeholders instead of guessed content.
- Treat rhetorical packaging as presentation, not evidence creation.
- Preserve consistency across names, dates, budget values, and metrics.
